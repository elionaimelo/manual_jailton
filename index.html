Fazer a solicitação de uma vaga de leito COVID

Eu como usuário NIR Solicitante quero acessar o sistema RegulaRN para realizar uma nova solicitação de vaga de leito COVID.

Realizar uma nova solicitação de vaga de leito COVID

Passo a passo:

  1. Seleciono o vínculo, o perfil "NIR - UPA 24H POTENGI(7923287)"
  2. Clico no botão "Iniciar"
  3. Vejo a página "Página Inicial"
  4. Clico na opção "Nova Solicitação"
  5. Vejo a página "Cadastrar Solicitação"
  6. Preencho o formulário com os dados "Médico"
  7. Clico no botão "Salvar e Prosseguir"
  8. Preencho o formulário com os dados "Pessoais"
  9. Clico no botão "Salvar e Prosseguir"
  10. Preencho o formulário com os dados "Endereço"
  11. Clico no botão "Salvar e Prosseguir"
  12. Preencho o formulário com os dados "Quadro"
  13. Clico no botão "Salvar e Prosseguir"
  14. Preencho o formulário com os dados "COVID-19"
  15. Clico no botão "Salvar e Prosseguir"
  16. Preencho o formulário com os dados "Sinais Vitais"
  17. Clico no botão "Salvar e Prosseguir"
  18. Preencho o formulário com os dados "Exames"
  19. Clico no botão "Salvar e Prosseguir"
  20. Preencho o formulário com os dados "SUPORTE"
  21. Clico no botão "Salvar e Prosseguir"
  22. Vejo a mensagem de confirmação "Solicitação cadastrada com sucesso."
