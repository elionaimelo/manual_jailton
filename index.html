Sobre o RegulaRN

O RegulaRN é um sistema desenvolvido para gerenciar a ocupação e disponibilidade de leitos médicos na rede de saúde do estado do Rio Grande do Norte.

Como acessar o RegulaRN?

O Sistema RegulaRN está acessível através do endereço https://regulacao.saude.rn.gov.br/.

Para acessar o sistema RegulaRN faz-se necessário antes realizar o autocadastro do NIR Solicitante. Para isso, faz-se necessário seguir os passos:

  1. Ter uma conta no Sabiá
  2. Fazer o autocadastro de Integrante NIR
  3. Aguardar o e-mail de validação do autocadastro por parte da central de regulação
  4. Acessar o sistema com a conta do Sabiá

Criando uma conta no Sabiá

Para realizar o cadastro no Sabiá, clique no link https://login.sabia.ufrn.br/entrar/. Após clicar no link, será aberta a página de login do Sabiá. Nessa página, será possível realizar o cadastro no link "Cadastre-se" ou alterar a senha no link "Esqueceu sua senha?"

Como realizar o autocadastro de Integrante do NIR?

Tendo usuário cadastrado no Sabiá, acesse o Sistema RegulaRN através do endereço https://regulacao.saude.rn.gov.br/ para fazer o autocadastro como NIR Solicitante.

O perfil de NIR solicitante poderá realizar solicitações de leitos de COVID-19 bem como atualizar as informações da solicitação.
